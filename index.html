💡 Configure o gabarito no Corretor OMR antes de usar a câmera.
⚠ Câmera bloqueada no celular
O celular exige que o site esteja em HTTPS ou na rede local para liberar a câmera.
A solução mais simples: acesse pelo Wi-Fi da escola usando o endereço do computador.
Como fazer em 3 passos:
1. No computador, abra o Prompt de Comando (Windows) ou Terminal (Mac/Linux)
2. Execute este comando na pasta onde está o arquivo:
python -m http.server 8080
3. Descubra o IP do computador:
Windows: ipconfig  |  Mac/Linux: ifconfig
Procure o número no formato 192.168.x.x
4. No celular (mesmo Wi-Fi), acesse:
http://192.168.x.x:8080/cartaoia.html
💡 Computador e celular precisam estar na mesma rede Wi-Fi. O servidor só precisa ficar ligado durante o uso.

Câmera bloqueada no celular

O celular exige que a página esteja em HTTPS para acessar a câmera. A solução mais fácil é acessar pelo Wi-Fi da escola — sem internet, sem custo.
No computador (Windows):
1. Abra o Prompt de Comando (tecla Windows → digite "cmd")
2. Cole e execute este comando:
python -m http.server 8080
3. Descubra o IP do computador: no mesmo terminal, digite ipconfig e anote o IPv4 (ex: 192.168.1.10)
4. No celular (mesmo Wi-Fi), abra o Chrome e acesse:
http://192.168.1.10:8080
5. Clique no arquivo cartaoia.html que aparece na lista.

No computador (Mac):
1. Abra o Terminal
2. Navegue até a pasta do arquivo: cd ~/Downloads
3. Execute: python3 -m http.server 8080
4. Descubra o IP: ifconfig | grep "inet 192"
5. No celular acesse http://IP:8080/cartaoia.html
